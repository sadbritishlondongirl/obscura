Из тишины — послание.

Obscura — мессенджер нового поколения на базе Nullo Trace Protocol (NTP).

Приватность. Эфемерность. Forward Secrecy. Метаданные исчезают в темноте.

  • State-Aware Ratchet — гибридный симметрично-асимметричный механизм
  • Эфемерные сообщения с таймером удаления (TTL)
  • Обмен ключами через QR-коды и nullo://ссылки
  • Групповые чаты с Double Ratchet (MLS-принцип)
  • Paradise Tree Protocol — маршрутизация через сеть прокладок
  • Локальное хранилище ратчетов .nullo/state.db
  • Оффлайн-доставка и восстановление сообщений
Скоро. Очень скоро.
