Возможности Obscura

Мессенджер нового поколения, который сочетает в себе безопасность, приватность и интеллект

Многоуровневое E2E шифрование

Triple-Layer Encryption: шифрование на транспортном, сессионном и контентном уровнях с использованием современных алгоритмов Curve25519, XSalsa20 и Poly1305.

  • Perfect Forward Secrecy защищает старые сообщения даже при компрометации ключей
  • Blake2b + Argon2: быстрый хеширование и защита от брутфорса
  • Готовность к пост-квантовой эре

Цифровые подписи

Все сообщения подписываются с использованием Ed25519, обеспечивая гарантию подлинности и целостности данных без централизованного хранилища ключей.

  • Многофакторная верификация собеседников
  • Защита от подмены сообщений и MITM-атак
  • Распределенная система доверия

Защита метаданных

Paradise Tree Protocol обеспечивает распределённую маршрутизацию сообщений через доверенные узлы с использованием принципов луковой маршрутизации.

  • Рандомизация размеров пакетов и задержек
  • Защита от анализа трафика и профилирования
  • Шифрование не только содержимого, но и маршрута

Zero-Knowledge Storage

Сервер ничего не знает о вашем содержимом благодаря Zero-Knowledge архитектуре и локальному шифрованию данных.

  • Самоуничтожающиеся сообщения с настраиваемыми таймерами
  • Безопасное удаление с физическим перезаписыванием
  • Максимальная локализация данных без облака

// Пример плагина для Obscura
class TranslatorPlugin {
    constructor() {
        // Инициализация с минимальными привилегиями
        this.permissions = [
            "message:read", 
            "message:transform"
        ];
        this.name = "Translator";
        this.version = "1.0.0";
    }

    async onMessage(message) {
        // Обработка сообщения
        if (message.hasCommand("/translate")) {
            const targetLang = message.getCommandParam("lang");
            const text = message.getCommandParam("text");
            
            const result = await this.translate(text, targetLang);
            
            return {
                type: "reply",
                content: result,
                metadata: {
                    originalLang: "auto",
                    targetLang: targetLang
                }
            };
        }
        return null;
    }

    async translate(text, lang) {
        // Безопасная имплементация перевода
        return await API.translation.process(text, lang);
    }
}

// Регистрация плагина с цифровой подписью
ObscuraPluginRegistry.register(
    new TranslatorPlugin(),
    "sig:ed25519:ZXhhbXBsZSBzaWduYXR1cmU="
);
